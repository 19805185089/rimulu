## 当前实现状态（基线）

### 1. 桌宠主交互
- 透明、无边框、置顶窗口（桌宠悬浮）。
- 点击史莱姆可触发弹跳动画与音效，并展开/收起环形菜单。
- 支持拖拽史莱姆移动窗口。
- 非交互区域支持鼠标穿透，交互区域可点击。
- 应用失焦后自动收起菜单。
- 当前环形菜单入口：`签到`、`技能`、`备忘`、`聊天`、`设置`。

### 2. 成长系统（签到 / EXP / 等级 / MP）
- 展示 `Lv`、`EXP`、`MP`、签到次数、连签信息。
- 每日签到（本地自然日）限制同日一次有效签到。
- 连签奖励（第 3/5/10 天）已实现。
- 经验累计支持连续升级。
- MP 按固定间隔自动恢复，随等级提升上限。
- 进度数据持久化到本地存储。

### 3. 技能系统
- 已实现技能：`水弹`、`火焰球`。
- 技能释放消耗 MP，MP 不足会提示。
- 技能释放获得随机 EXP，并可能触发升级。
- 已实现技能施法与命中特效动画。
- 技能菜单已重构为长方形面板（位于史莱姆上方，支持拖拽）。
- 技能图标底色按元素配色（水系蓝、火系橙红）。
- 技能特效生成锚点已调整到史莱姆中心。

### 4. 备忘录系统
- 支持新增、编辑、保存、删除备忘。
- 支持完成勾选、完成时间和耗时字段记录。
- 支持提醒时间设置（`datetime-local`）。
- 到点触发应用内提醒，并尝试发送系统通知。
- 备忘录面板支持拖拽定位、固定/取消固定。
- 已保存备忘持久化到本地存储。

### 5. 吞噬（拖拽删除到回收站）
- 支持将文件/目录拖拽到史莱姆嘴部热区。
- 命中热区后弹出确认框，确认后执行处理。
- 通过 Tauri Rust 命令批量移入系统回收站（非永久删除）。
- 成功与失败均有提示反馈。
- 吞噬入口已迁移到“设置面板”中的开关项，可动态启用/禁用。
- 设置项旁提供 `?` 悬浮说明（解释吞噬功能用途）。

### 6. Tauri / Rust 能力
- 已接入插件：`notification`、`opener`。
- 已暴露命令：`swallow_delete_to_trash(paths)`。
- capability 已配置窗口拖拽、忽略鼠标事件、通知权限等能力。

### 7. LLM 聊天与配置（当前）
- 聊天面板已完整接入真实接口，不再是占位态；支持多轮消息、发送中状态、失败提示。
- 已支持 OpenAI 兼容 `chat/completions` 请求链路，含流式与非流式解析。
- 错误处理已增强：请求失败会弹出完整错误详情（含 endpoint/request/raw error 片段），不再仅依赖短暂 toast。
- 配置面板支持 LLM 启用、最小兼容、流式响应、服务商、Base URL、模型、API Key、系统提示词及高级参数。
- 已加入“上下文轮数（问答）”配置（默认 20，范围 1-50），请求与本地聊天缓存按该配置进行裁剪。
- 聊天面板提供“清空上下文”操作，点击后清空当前会话历史缓存。

## 已知待完善项

- 吞噬流程缺少操作历史/撤回等更完整交互。
- 当前上下文仅做窗口内短期缓存与拼接，尚未接入“长期记忆/检索记忆/摘要记忆”机制。
- 不同网关对 OpenAI 兼容格式存在差异，仍需继续做响应结构与错误回退的兼容性收敛。
- 项目尚未配置自动化测试（单测/集成测试）。
